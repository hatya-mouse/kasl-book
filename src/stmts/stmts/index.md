# Statements

Statements can be used inside functions, and creates the actual execution logic.

| Statement | Description | Link |
| --- | --- | --- |
| Block | Creates a scope. | [Block](./block.md) |
| var | Declares a local variable. | [Var](./var.md) |
| let | Declares a local constant. | [Let](./let.md) |
| = | Assigns a value to the variable or its field. | [Assign](./assign.md) |
| if / if-else | Conditionally execute the statements. | [If](./if.md) |
| return | Exits from the function with a return value. | [Return](./return.md) |
| loop | Creates a loop that will be executed specific times. | [Loop](./loop.md) |
